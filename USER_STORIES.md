# User Stories - Academic Submission App

## 1. User Registration
As a new user, I want to be able to create an account so that I can access the application and save my information.

## 2. User Login
As a registered user, I want to be able to log in to the application so that I can access my personalized content.

## 3. View Academic Posts
As a user, I want to be able to view a list of academic posts on the home screen so that I can browse available content.

## 4. View Post Details
As a user, I want to be able to view the full details of a post so that I can read the complete content.

## 5. Save Favorites
As a user, I want to be able to save posts to my favorites so that I can easily access them later.

## 6. Manage Favorites
As a user, I want to be able to view and manage my saved favorites so that I can remove posts I no longer need.

## 7. App Settings
As a user, I want to be able to customize app settings like dark mode and notifications so that I can personalize my experience.

## 8. Push Notifications
As a user, I want to receive push notifications from the app so that I can stay informed about important updates.

## 9. Logout
As a logged-in user, I want to be able to log out of the application so that my account is secure when I leave the device.
